# 📊 Loan Risk & Portfolio Analysis Dashboard (Power BI)

## 🚀 Project Overview

This project presents an end-to-end **Loan Portfolio & Risk Analysis Dashboard** built in Power BI.
It focuses on analyzing loan performance, identifying high-risk segments, and enabling data-driven decision-making for financial institutions.

The dashboard combines **KPI tracking, segmentation, trend analysis, and risk classification** into a single interactive experience.

## 🎯 Project Objectives

* Analyze loan portfolio performance
* Identify high-risk borrower segments
* Track loan disbursement trends over time
* Evaluate default behavior across multiple dimensions
* Build an interactive and user-friendly dashboard

---

## 📊 Key Insights

* 📉 Lower-grade loans (E–G) show significantly higher default rates
* 💰 Low-income borrowers are more likely to default
* 🏠 Renters have higher default rates compared to homeowners
* 📌 Certain loan purposes (e.g., small business, debt consolidation) carry higher risk
